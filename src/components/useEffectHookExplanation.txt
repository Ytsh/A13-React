useEffect hook.

- it is a react hook that allows you to run side effects in you function component.
- fetching data from API, directly updating the dom, setting up subscription and timers, 
    listening to events.

    -Asynchronous programming, 
    - mount, inbetween, unmount
    - componentDidMount, CompoentDidUpdate, CompoentDidUnmount

            //your code here
